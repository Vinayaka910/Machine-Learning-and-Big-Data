# Project Overview

This repository contains the implementation and supporting notebooks for the 7006SCN Machine Learning and Big Data coursework. The project applies PySpark-based machine learning techniques to a large-scale dataset, demonstrating the complete machine learning lifecycle from data ingestion and preprocessing through model development, evaluation, distributed computing optimisation, and business intelligence visualisation.

The work follows the coursework structure and showcases the use of distributed data processing, machine learning model development, explainable AI techniques, and Tableau-based storytelling.

# Project Objectives
Perform large-scale data ingestion and preprocessing using PySpark.
Apply feature engineering techniques to improve model performance.
Develop and compare multiple machine learning models.
Implement distributed computing optimisation techniques.
Evaluate model performance, robustness, and explainability.
Create interactive Tableau dashboards to communicate business insights.
